# scoop-bucket

Add this bucket via:

```powershell
scoop bucket add taiki-e https://github.com/taiki-e/scoop-bucket
```

Remove this bucket via:

```powershell
scoop bucket rm taiki-e
```

With the following commands, you can install the latest version of each package:

```powershell
scoop install cargo-hack
scoop install cargo-llvm-cov
scoop install cargo-minimal-versions
scoop install parse-changelog
```
